Step 0: Gearing Up for Go!

"Before we write our first line of Go magic," Froggy advises, "make sure you have Go installed on your computer! You can grab it from the official Go download page."

Froggy also mentions, "Go programs are organized into packages, and related packages are grouped into modules. We'll create a new module for our DNS Frog Gopher project soon!"

Froggy's Setup Tip: If you're new to Go, the official "Getting Started" tutorial is a great way to set up your environment and write a 'Hello, World!' program. Don't worry, we'll go through the basics together too!
