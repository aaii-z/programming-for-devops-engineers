Project Overview: DNS Frog Gopher

Our DNS Frog Gopher tool will be a command-line program that you can run from your terminal. By the end of this adventure, it will be able to:

  • Take a domain name (like google.com) as input.
  • Look up different types of DNS records (A, MX, CNAME, TXT).
  • Display the results in a clear, user-friendly way.

Froggy says, "This is a super practical tool for any DevOps engineer! Let's start building!"
