üß™ Time to Get Our Hands Muddy: Practice Challenges!

Froggy believes the best way to learn is by doing! Here are some challenges for you to try:

  • Automate backup and cleanup tasks on a server using shell scripts.

    Example script for automating backups:

    #!/bin/bash
# Simple backup script example
SRCDIR="/var/www/html"
DSTDIR="/mnt/backups/daily"
FILENAME="backup_$(date +%Y-%m-%d_%H-%M-%S).tar.gz"

echo "Starting backup of ${SRCDIR}..."
tar -czf "${DSTDIR}/${FILENAME}" "${SRCDIR}"

if [ $? -eq 0 ]; then
  echo "Backup successful: ${DSTDIR}/${FILENAME}"
else
  echo "Backup failed!"
fi
  • Create a health check script to monitor system resources (e.g., disk space, CPU usage).
