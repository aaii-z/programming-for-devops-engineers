Project Overview: Log Detective Froggy

Put on your detective hat with Froggy! In this project, we'll create a Bash script that sifts through log files to find important information, like errors or specific events.

Our "Log Detective Froggy" script will aim to:

  • Accept a log file path as input.
  • Search for specific keywords or patterns (e.g., "ERROR", "WARN", IP addresses).
  • Filter and extract relevant lines.
  • Optionally, count occurrences or generate a simple summary.

Froggy's Blueprint: Log Detective

Our script will read log files, process them, and output useful summaries.

This project will help us master text processing tools like grep, awk, and sed, along with loops and conditional logic in Bash.
